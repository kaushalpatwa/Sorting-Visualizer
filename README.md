# Sorting Visualizer

An interactive Sorting Visualizer built with HTML, CSS, JavaScript, and Bootstrap. This project demonstrates Bubble Sort, Selection Sort, Insertion Sort, Quick Sort, and Merge Sort using animated visualizations.

## Features
- Generate random arrays
- Adjustable array size
- Adjustable sorting speed
- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## How to Run
1. Clone the repository.
2. Open `index.html` using Live Server in VS Code.
3. Choose a sorting algorithm to visualize.
